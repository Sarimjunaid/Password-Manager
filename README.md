A full-stack Password Manager application built with React, Tailwind CSS, Express.js, and MongoDB.
This project allows users to securely store, manage, and organize their website credentials through a clean and responsive interface.

Features
Add new passwords
View saved credentials
Edit existing passwords
Delete passwords
Responsive UI with Tailwind CSS
REST API using Express.js
MongoDB database integration
Tech Stack
Frontend: React, Tailwind CSS
Backend: Express.js, Node.js
Database: MongoDB
